# NAME IT SCENTS - Luxury Perfume E-commerce Website

A modern, responsive one-page e-commerce website for "NAME IT SCENTS", a luxury perfume brand. This website showcases the brand's collection with an elegant design and professional user experience.

## 🌟 Features

### Design & User Experience
- **Modern & Professional Design**: Clean, elegant layout with luxury aesthetics
- **Fully Responsive**: Optimized for all devices (desktop, tablet, mobile)
- **Smooth Animations**: CSS animations and transitions for enhanced user experience
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic content

### Sections
1. **Home/Hero Section**: Eye-catching banner with call-to-action
2. **Featured Products Carousel**: Showcasing premium perfumes with auto-advance
3. **About Section**: Brand story and key features
4. **Gallery Section**: Filterable perfume collection by categories
5. **Contact Section**: Contact form and business information
6. **Footer**: Social links, quick navigation, and newsletter signup

### Interactive Features
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Product Carousel**: Manual and auto-advancing carousel with touch/swipe support
- **Gallery Filtering**: Filter perfumes by category (Floral, Woody, Oriental, Fresh)
- **Contact Forms**: Functional contact and newsletter forms with validation
- **Add to Cart**: Interactive product cards with cart functionality
- **Smooth Scrolling**: Seamless navigation between sections

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript (ES6+)**: Interactive functionality
- **Font Awesome**: Icons for enhanced visual appeal
- **Google Fonts**: Typography (Playfair Display & Poppins)

- **Lazy Loading**: Images load as they come into view

## 🔧 Customization

### Adding New Products
1. Add new carousel slides in the HTML
2. Include corresponding gallery items
3. Update images and product information

### Modifying Colors
1. Update CSS custom properties in `styles.css`
2. Maintain color consistency across sections

### Adding New Categories
1. Add filter buttons in the gallery section
2. Update JavaScript filtering logic
3. Add corresponding gallery items with proper data attributes
